# SQL Query Generator

## Description

This project converts natural language queries into SQL queries using Python and Natural Language Processing.

## Technologies Used

* Python
* NLP
* Google Colab
* Gradio

## How to Run

2. Run all cells
3. Enter a natural language query
4. The system will generate the SQL query
